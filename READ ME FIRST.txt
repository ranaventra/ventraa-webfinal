VENTRAA — STATIC build with NATIVE GHL inline form embeds (Netlify-ready · June 2026)
=====================================================================================

LOCAL PREVIEW
  Open ventraa-homepage.html. Empty form areas show a dashed "embed here" box.

WHAT CHANGED
  The 8 form field areas are now Go High Level INLINE EMBED SLOTS.
  Headings, the dark branded form cards, spacing, layout and the consent line
  are all unchanged — only the input fields were replaced by a slot.
  Removed: ghl-config.js, forms.js (no webhook / no config / no API / no serverless).

HOW TO CONNECT EACH FORM (per slot)
  1. In GHL: Sites > Forms > open (or build) the form named below.
  2. Click "Integrate Form" > the "Inline" tab > Copy the code.
  3. Open the page, find the matching <div class="ghl-embed"> (clear HTML comment
     inside it), and paste the snippet on the line below the comment.
  A native GHL form creates/updates the contact automatically — no webhook needed.
  For team alerts: the form's own Notifications setting, or a standard (non-premium)
  "Form Submitted" workflow trigger.

DEPLOY TO NETLIFY
  Drag this whole folder onto app.netlify.com (or connect a repo and set the
  publish directory to this folder). The included _redirects file serves the
  homepage at "/". robots.txt and sitemap.xml are included.
  Tip: keep Netlify "Pretty URLs" on so /lgs serves lgs.html (matches the
  canonical tags and sitemap).
